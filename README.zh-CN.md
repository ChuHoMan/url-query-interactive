## 安装

确保先安装依赖:

```bash
pnpm install
```

## 启动开发服务器

启动开发服务器到 http://localhost:3000

```bash
pnpm run dev
```

## 生产

构建生产应用：

```bash
pnpm run build
```

本地预览生产应用:

```bash
pnpm run preview
```

Checkout the [deployment documentation](https://v3.nuxtjs.org/docs/deployment) for more information.
